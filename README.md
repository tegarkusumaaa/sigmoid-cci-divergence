# Sigmoid CCI 100 - Divergence & Momentum Weakening Indicator
<img width="1818" height="887" alt="XAUUSD_2026-07-21_11-25-54" src="https://github.com/user-attachments/assets/fbda2002-1160-45b0-997f-58b1f2d2dccf" />

An advanced Pine Script v5 technical indicator that normalizes the unbounded Commodity Channel Index (CCI) into a bounded 0–100 scale using a non-linear Logistic Sigmoid transformation. Includes automated pivot detection for classic divergences and extreme saturation weakening setup identification.

## 🚀 Key Features
- **Sigmoid Normalization Engine:** Compresses extreme tail variations of raw CCI values into a bounded range (0 to 100), eliminating outlier distortion.
- **Automated Divergence Detection:** Built-in pivot algorithms identify classic Bullish/Bearish divergences between price high/low swings and oscillator momentum.
- **Saturation Weakening Detection:** Detects high-probability reversal setups when price makes a higher high/lower low while the oscillator flattens at extreme saturation thresholds.
- **Clean UI & Grouping:** Fully customizable inputs categorized logically with visual zone highlights for fast chart reading.

## 🛠️ Usage & Installation
1. Copy the code from `SigmoidCCI.pine` in this repository.
2. Open TradingView, launch the **Pine Editor** at the bottom panel.
3. Paste the code, save it, and click **Add to chart**.
